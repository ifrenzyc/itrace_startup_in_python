## 选拔赛题目二：
**运行：**
``` shell
$ pip install Flask
$ FLASK_APP=section02.py flask run
```

## ORM 集成
集成了 SQLAlchemy 包，是 Python 语言的一个 ORM 包，支持多种数据库。

### 安装

``` shell
$ pip install SQLAlchemy flask-sqlalchemy pymysql
```

例子接口：
[http://127.0.0.1:5000/cust/get](http://127.0.0.1:5000/cust/get)
